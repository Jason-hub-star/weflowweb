## Visual Direction

키워드:

- dark charcoal
- fresh startup
- mint/cyan glow
- ice-blue clarity
- lime success accent
- premium but lightweight
- Devfive-inspired structure

핵심:

- 배경은 차콜/블랙 계열
- 포인트는 민트, 시안, 아이스블루, 라임
- 보라/주황/갈색 계열은 피한다.
- 어둡지만 무겁지 않고, “빠르게 런칭하는 SaaS 스타트업” 느낌을 준다.

## Non-Negotiables

- Devfive 원본 코드/이미지/캐릭터를 복제하지 않는다.
- Devfive의 구조적 장점만 WEFLOW에 맞게 재해석한다.
- Flow Guide 캐릭터는 보조 포인트로만 사용한다.
- 모바일 375px에서 텍스트, 카드, 폼, CTA가 넘치지 않아야 한다.
- 최종 전환은 문의, 견적요청, 카카오톡 상담이다.
- 쇼핑/결제/회원가입 기능은 필요 없다.
- 병원/의료기술/치료/시술/의료 효과 관련 표현과 이미지는 금지한다.
