# PRD — WEFLOW 공식 사이트 (1차)

작성: 2026-05-29 · 출처 잠금: DEC-001 ~ DEC-032

---

## 1. Problem

소상공인·자영업 대표는 홈페이지/랜딩페이지를 만들고 싶지만:
- 디자인 외주는 비싸고 운영까지 연결되지 않음
- 모바일·검색 노출·광고 운영을 따로 따로 의뢰해야 함
- 만든 후 문의가 이어지지 않음

WEFLOW는 **제작 + 광고 + 운영 + 관리**를 한 번에 묶어 "**문의로 이어지는 홈페이지**"를 만든다.

기존 `weflow-web.vercel.app`(현재 공개 프리뷰)는 모바일 동선이 불편하고 전환 구조가 약하다 → 리디자인 필요.

---

## 2. Users

- **주요 타깃**: 30대 남성 중심, 30~40대 대표/실무자까지 확장
- **상황**: 신규 창업 또는 기존 사업의 디지털 전환
- **의사결정 흐름**: 검색 → 회사 소개 → 가격 → 사례 → 문의/상담

페르소나 행동:
- 모바일에서 빠르게 둘러보고 카카오톡으로 1차 문의
- 데스크톱에서 가격·사례를 비교 후 무료진단 신청
- 후기를 신뢰의 마지막 근거로 사용

---

## 3. Scope — 1차

- 정적/SSG·ISR 기반 마케팅 사이트
- 17 라우트 (마스터 §7)
- 무료진단·예약 폼 → Resend + Google Sheets + Vercel BotID
- SEO·픽셀·OG 풀스택
- 모바일 375 우선 디자인

## 4. Non-Scope — 2차

- 관리자 대시보드 10화면
- Neon Postgres / Clerk 인증
- 후기 사용자 직접 작성
- 영어/일본어 i18n
- 다크모드

## 5. Brand Copy Lock

- 헤드라인: `문의로 이어지는 홈페이지를 만듭니다`
- 서브헤드: `홈페이지 제작부터 광고 연동·운영 관리까지, 단순 제작이 아닌 문의 구조까지 설계합니다.`
- 아이브로우: `랜딩&홈페이지 제작 · 광고 운영 · 검색 등록 · 맞춤형 웹 솔루션`
- CTA: Primary `무료 진단 신청` / Secondary `성공 사례 보기` / Tertiary `랜딩 페이지 사례`
- 베네핏 태그: `케어 플랜` · `3~7일 빠른 제작` · `합리적 비용`

## 6. 톤 & 매너 Lock

- 30~40대 대표가 신뢰할 수 있는 **밝고 친근한 프리미엄 미니멀**
- 신뢰감 · 빠른 실행 · 전환 중심 · 운영 관리 · 데이터/성과

## 7. 금기 Lock

- 병원·의료기술·치료·시술·의료 효과 표현·이미지
- `SEO 상단등록`·`검색 상단 노출` 보장형 표현
- 일반 SaaS 템플릿스러운 추상성
- 사이버펑크·과한 미래형
- 광고 전단 같은 저렴함

## 9. 필수 폼 필드 Lock

### 무료진단/견적 문의
| 필드 | 필수 | 형식 |
|---|---|---|
| 이름 | ✓ | text |
| 연락처 | ✓ | tel |
| 제작종류 | ✓ | select (랜딩/홈피/랜딩&홈피/기타-WEFLOW케어) |
| 업종 | 선택 | text |
| 추가요청사항 | 선택 | textarea |
| 개인정보 수집 및 상담 동의 | ✓ | checkbox |

상태 카피:
- success: `접수 완료! 빠른 시간 내에 연락드리겠습니다.`
- validation: `개인정보 수집 및 상담 동의가 필요합니다.`
- error: `잠시 후 다시 시도해 주세요.`

버튼: `무료진단 후 견적 받기`

### 예약
무료진단 필드 + `날짜` · `시간대(select)` · `직접 입력 시간대(text)`

## 10. Success Criteria

- 모바일 375 — 모든 CTA·카드·폼 텍스트 넘침 없음
- Lighthouse Perf ≥ 90, A11y ≥ 95, SEO 100
- `pnpm build` 0 error / 0 warning
- 폼 제출 → Resend 이메일 수신 + Sheets 1행 추가 + BotID 우회 거부 검증 통과
- `MISSING-AND-UNIMPLEMENTED.md` 0건
- 의료/병원 키워드 grep 0건
